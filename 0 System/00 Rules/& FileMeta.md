## Prefix

- `_` : Template
- `&` : System
- `}` : Inbox
- `~` : Literature Card
- `'` : Main Card
- `)` : Keyword
- `%` : Structure Card
- `$` : Project
- `;` : Area
- `@` : Character
- `!` : Output
- `+` : Test
- `^\d{4}` : Periodic (Year)
- `^(?!\d{4})[^_&}~'%$;@!+]` : Terms(용어)

## Extension Prefix

- `.post` : Post
- `.presentation` : Presentation
- `.script` : Script
- `.drawing` : Drawing
- `.excalidraw` : Excalidraw
- `.kanban` : Markdown
